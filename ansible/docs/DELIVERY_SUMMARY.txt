🆘 QUICK TROUBLESHOOTING

Problem                         Solution
──────────────────────────────────────────────────────────────────────
Ansible not found               make ansible-setup
Cluster already exists          kind delete cluster --name hookdrop
Permission denied               sudo chown -R $(whoami):staff ~/.docker
Python version error            python3 --version (need 3.8+)
ArgoCD password lost            kubectl -n argocd get secret argocd-initial-admin-secret -o jsonpath="{.data.password}" | base64 -d
Kyverno blocking pods           Update manifests (no 'latest' tags, add resource limits)
Network timeout                 Check internet, manifests need to download
